# Бот-секретарь для Телеграма — запуск за ~15 минут

Ловит задачи и встречи в твоих рабочих чатах → шлёт тебе в ЛС → напоминает,
пока не закроешь → каждый день делает отчёт.

---

## Шаг 1. Создай бота (2 мин)
1. В Телеграме напиши **@BotFather** → `/newbot` → задай имя и юзернейм.
2. Он выдаст **токен** вида `12345:AAH...` — скопируй.
3. Там же отключи приватный режим, иначе бот не увидит сообщения в группах:
   `/setprivacy` → выбери своего бота → **Disable**.

## Шаг 2. Настрой файл .env (2 мин)
1. Переименуй `.env.example` → `.env`.
2. Вставь токен в `TELEGRAM_BOT_TOKEN`.
3. (Рекомендую) вставь `ANTHROPIC_API_KEY` с https://console.anthropic.com —
   с ним бот понимает смысл и даты, без него ловит по ключевым словам.
4. Поправь `TIMEZONE` и время отчёта под себя.

## Шаг 3. Запусти (на компе — для теста)
```bash
pip install -r requirements.txt
python bot.py
```
Затем в Телеграме напиши боту **`/start`** в личке — это привязывает тебя
как владельца (только тебе он будет слать задачи и отчёты).

## Шаг 4. Добавь бота в рабочие чаты
Добавь его участником в нужные группы. Дальше он сам ловит:
- сообщения, где тебя упомянули (`@ты`),
- ответы на твои сообщения,
- сообщения людей из «списка наблюдения» (`/watch @artur`),
- либо вообще все, если в `.env` поставить `SCAN_ALL=true`.

## Шаг 5. Чтобы работал круглосуточно
На компе он живёт, только пока открыт терминал. Для постоянной работы
(напоминания и отчёты приходят, даже когда комп выключен) — подними его
на маленьком всегда-включённом сервере (VPS за ~3–5$/мес). Типовой путь:
```bash
# на сервере
git clone <твой репозиторий>  # или просто скопируй файлы
pip install -r requirements.txt
# запуск как служба, чтобы поднимался сам:
sudo tee /etc/systemd/system/secretary.service >/dev/null <<'EOF'
[Unit]
Description=TG secretary bot
After=network.target
[Service]
WorkingDirectory=/root/bot
ExecStart=/usr/bin/python3 /root/bot/bot.py
Restart=always
[Install]
WantedBy=multi-user.target
EOF
sudo systemctl enable --now secretary
```
⚠️ Бесплатные хостинги, которые «засыпают», не подойдут — для напоминаний
процесс должен быть живым постоянно.

## Команды бота
| Команда | Что делает |
|---|---|
| `/start` | привязать тебя как владельца |
| `/tasks` | показать открытые задачи с кнопками |
| `/report` | прислать отчёт прямо сейчас |
| `/watch @artur` | всегда отслеживать сообщения этого человека |
| `/unwatch @artur` | перестать |
| `/repeat 20` | напоминать каждые N минут |

## Честно про ограничения
- Бот видит сообщения только в чатах, **где он состоит**, и только **после**
  того как его добавили (старую переписку не читает).
- В общих чатах коллеги **увидят бота** в списке участников. Учитывай это.
- «Долбить, пока не нажмёшь Выполнено» работает в рамках тихих часов
  (по умолчанию ночью 22:00–08:00 молчит — это настраивается).
- Личный Google-календарь пока не подключён — это отдельный шаг (доступ к Google),
  добавляется поверх готового бота.
